# UNFOLD Readiness

This repository is the public launchpad and reviewer-facing technical home for PassPal.

## Current purpose

It is designed to make the current wedge easier to understand for:

- ecosystem reviewers
- interoperability partners
- verifier / relying-party evaluators
- pilot and partnership discussions

## What this repo currently provides

- product overview
- current wedge explanation
- flow summary
- receipt model
- multilingual documentation entry points
- shared place for future sample payloads and API preview files

## Current boundary

This repository is not the full internal production source of truth.

It is the public-facing launchpad layer for clearer review and discussion.

## Packaging rule

Ecosystem-specific packaging may evolve later depending on onboarding requirements.
