# Product Overview

PassPal is a hosted trust decision interface.

It is designed to:

- request the minimum useful proof
- evaluate trust-sensitive workflow inputs
- return a decision-ready output
- help relying parties act on trust-sensitive results

## What PassPal does

PassPal helps transform a targeted proof request into a business-readable result.

Instead of stopping at identity collection, it aims to support the actual decision step:
- continue
- ask for more proof
- escalate
- stop

## What PassPal is not

PassPal is not:

- another wallet
- a raw document vault
- a generic profile directory
- a vague score-only product

## Product stack

### PassPal
The trust decision interface.

### Passpod
The trust protocol and orchestration layer behind PassPal.

### DIDX
The trust registry and public trust surface layer.
